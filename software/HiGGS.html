Hamiltonian Gauge Gravity Surveyor (HiGGS)

Go to the GitHub repository for download.

HiGGS provides tools for the Hamiltonian constraint analysis (canoncical analysis or Dirac-Bergmann algorithm) of gravity with spacetime curvature and torsion. HiGGS can be used on a desktop PC, but it is parallelised for theory surveys on clusters and supercomputers.
